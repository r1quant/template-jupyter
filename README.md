
## Steps 

- Install UV: https://docs.astral.sh/uv/#installation
- Open terminal, go to project (e.q: `cd /jupyter-starter`)
- Execute `uv sync`
- Open VSCode, open the file `basic.ipynb`
- Click on `Select Kernel` then select 'Python Environments.. > Python (./venv/bin/active)'
- Click on `Run All`
